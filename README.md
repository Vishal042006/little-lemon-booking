# 🍋 Little Lemon Booking App

A React-based web application that allows users to book a table at the Little Lemon restaurant. This project demonstrates front-end development skills including form handling, validation, accessibility, and unit testing.

---

## 🚀 Features

- 📅 Table booking form  
- ✅ Input validation (name, date, guests)  
- ⚠️ Error handling with user-friendly messages  
- 🎯 Accessible form elements (labels, aria attributes)  
- 🧪 Unit testing using React Testing Library  
- 📱 Responsive design  

---

## 🛠️ Technologies Used

- React.js  
- JavaScript (ES6)  
- HTML5 & CSS3  
- React Hooks (`useState`)  
- Jest & React Testing Library

## ⚙️ Installation & Setup

1. Clone the repository:

git clone https://github.com/Vishal042006/little-lemon-booking.git


2. Navigate into the project:

cd little-lemon-booking


3. Install dependencies:

npm install


4. Start the development server:

npm start


---

## 🧪 Running Tests


npm test
